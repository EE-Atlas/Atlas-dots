# Atlas-Dots

## Installation
### Step 1: Install `dotify`
```
git clone https://github.com/encore-ecosystem/dotify.git
cd dotify
uv sync
source .venv/bin/activate
```

### Step 2: Apply config
```
git clone https://github.com/EE-Atlas/Atlas-dots.git
cd Atlas-dots
dotify apply
```
